AudioMaster Pro Studio

เครื่องมือ Mastering ออนไลน์ที่ทำงานบนเบราว์เซอร์ของคุณโดยตรง
ใช้ Web Audio API ในการประมวลผลเสียงแบบ Real-time

เริ่มต้นใช้งานอย่างรวดเร็ว

1

อัพโหลดไฟล์เสียง

คลิกปุ่ม Upload Audio หรือลากไฟล์เสียงวางที่ช่อง Waveform รองรับไฟล์ WAV, MP3, FLAC, OGG

2

เลือก Genre Preset

เลือกแนวเพลงจาก Sidebar (Desktop) หรือแท็บ Genres (มือถือ) ระบบจะปรับค่า EQ และ Compressor อัตโนมัติให้เหมาะกับแนวเพลงนั้น

3

ปรับแต่งเสียงด้วย Sliders

ใช้ Slider ทั้ง 10 ตัวในส่วน Mastering Controls เพื่อปรับค่าต่างๆ แบบ Real-time เช่น Bass, Treble, Compressor ฯลฯ

4

ฟังผลลัพธ์แบบ Real-time

กดปุ่ม Play เพื่อฟังเสียงที่ปรับแล้ว ดู Spectrum Analyzer และ EQ Curve เปลี่ยนแปลงได้แบบเรียลไทม์

5

ปรับแต่งละเอียดด้วย EQ Handles

บนกราฟ EQ Response Curve มี 7 จุดกลม (Handles) ให้ลากขึ้น-ลงปรับค่าได้โดยตรง ค่าจะ Sync กับ Slider ด้านล่างทันที

6

Export ไฟล์เสียง

เมื่อพอใจผลลัพธ์แล้ว กดปุ่ม Export ระบบจะ Render เสียงพร้อมเอฟเฟกต์ทั้งหมดแล้วดาวน์โหลดเป็นไฟล์ WAV 16-bit ให้ทันที

ฟีเจอร์หลัก

Drag & Drop Upload

ลากไฟล์เสียงวางที่ช่อง Waveform ได้เลย หรือคลิกปุ่ม Upload Audio เพื่อเลือกไฟล์

Waveform Player

แสดงรูปคลื่นเสียงแบบละเอียด มี Seek Bar สำหรับกระโดดไปยังจุดต่างๆ ของเพลงได้

Spectrum Analyzer

แสดงกราฟความถี่เสียงแบบ Real-time 64 แท่ง พร้อม Peak Indicator สี Mint เมื่อเสียงดังเกิน 70%

EQ Response Curve

แสดงเส้นโค้ง EQ 7 Band อัพเดทแบบ Real-time มี Drag Handles ให้ลากจุดกลมบน Curve เพื่อปรับค่าได้โดยตรง พร้อมปุ่ม Reset EQ คืนค่าทุก Band เป็น 0 dB ทันที

35 Genre Presets + Reset

เลือกแนวเพลงได้ 35 แนว ตั้งแต่ Pop, Rock, EDM ถึง Afrobeats, Shoegaze และ Grunge แต่ละ Preset ปรับค่า EQ + Compressor อัตโนมัติ ปรับเพิ่มเติมแล้วอยากกลับ? กด Reset Preset คืนค่ากลับไป Preset เดิมได้ทันที

Export WAV

ส่งออกเสียงที่ Master แล้วเป็นไฟล์ WAV 16-bit คุณภาพสูง รวมเอฟเฟกต์ทั้งหมดที่เปิดใช้ พร้อมดาวน์โหลดทันที

รายละเอียด Controls

Gain

ปรับความดังรวมของเสียง ย่าน -24 ถึง +24 dB เพิ่ม/ลด 0.5 dB ต่อครั้ง

Bass

Low Shelf Filter ที่ 150 Hz ปรับความดังย่านเบส -12 ถึง +12 dB

Mid

Peaking Filter ที่ 1 kHz (Q=0.7) ปรับความดังย่านกลาง -12 ถึง +12 dB

Treble

High Shelf Filter ที่ 3 kHz ปรับความดังย่านสูง -12 ถึง +12 dB

Air

Peaking Filter ที่ 12 kHz เพิ่มความสว่าง/ความโปร่งให้เสียง -6 ถึง +6 dB

Compressor

Dynamic Compressor ปรับอัตราส่วนบีบอัด 0-100% (Ratio 1:1 - 12:1) ลดช่วง Dynamic Range

Saturation

WaveShaper จำลองเสียง Saturation/Harmonic Distortion 0-100% ให้ความอบอุ่นแก่เสียง

Stereo Width

ปรับความกว้างของ Stereo Image 0% (Mono) ถึง 200% (Extra Wide)

Noise Gate

กำหนดระดับ Noise Floor -80 ถึง 0 dB เสียงที่เบากว่าจะถูกตัดออก

De-esser

ลดเสียง Sibilance (เสียง ส/ช) 0-100% ช่วยให้ Vocal นุ่มขึ้น

เคล็ดลับการ Mastering

①

เริ่มจาก Preset ก่อน — เลือก Genre Preset ที่ใกล้เคียงที่สุด แล้วค่อยปรับแต่งเพิ่มเติมจะเร็วกว่าเริ่มจากศูนย์

②

อย่าเพิ่ม Bass มากเกิน — +6 dB น่าจะพอสำหรับส่วนใหญ่ เพิ่มมากไปจะทำให้ Mix รกและผิดเพี้ยน

③

Compressor ช่วยกระชับเสียง — ค่า 40-60% เหมาะสำหรับเพลงส่วนใหญ่ ถ้าเกิน 70% เสียงจะถูกบีบจนไม่เป็นธรรมชาติ

④

Air เพิ่มความโปร่ง — ใช้ +2 ถึง +4 dB สำหรับ Vocal หรือ Acoustic จะช่วยให้เสียงเปิดและสว่างขึ้น

⑤

Reset Preset กลับไปเดิมได้ — ปรับเพิ่มเติมแล้วไม่ชอบ? กดปุ่ม Reset Preset คืนค่าทุก Slider กลับไปที่ Preset ของ Genre นั้นๆ ได้ทันที ไม่ต้องเริ่มใหม่

⑥

ลาก Handle บน EQ Curve — วิธีเร็วสุดคือลากจุดกลมบนกราฟ EQ แทนการใช้ Slider ทีละอัน ปรับได้ทั้ง 7 Band ในที่เดียว

⑦

Reset ได้เสมอ — กดปุ่ม Reset All เพื่อคืนค่าทุกตัวกลับเป็น Default หรือ Reset EQ เพื่อคืนเฉพาะ 7 Band EQ ไม่ต้องกลัวทำพัง

⑧

Export เมื่อพอใจ — ปรับจนพอใจแล้วกด Export ได้เลย ไฟล์จะ Render ผ่านเอฟเฟกต์ทั้งหมดที่เปิดอยู่

การใช้งานบนมือถือ

Bottom Navigation

ใช้แถบนำทางด้านล่างสำหรับสลับไปยังส่วนต่างๆ: Player, Genres, Controls, Analyzer

Genre Drawer

แตะแท็บ Genres หรือปุ่มเมนู เพื่อเปิดช่องเลือก Genre แบบ 3 คอลัมน์ ขึ้นมาจากด้านล่าง

Touch Sliders

สไลด์นิ้วบน Slider เพื่อปรับค่า รองรับการสัมผัส แตะแล้วลากได้ทุกตัว

Responsive Layout

การ์ดและ Slider จะจัดเรียงใหม่อัตโนมัติให้พอดีกับหน้าจอขนาดเล็ก

ปรับเสียงพอแล้ว ทำอะไรต่อ?

1. Export — เลือกฟอร์แมตและดาวน์โหลด

กดปุ่ม Export จะเปิดหน้าต่างเลือกฟอร์แมตไฟล์ มีให้เลือก 6 ฟอร์แมต:

WAV 16-bit CD Quality — ไฟล์ขนาดเล็กสุดในบรรดา WAV
WAV 24-bit Studio Quality — คุณภาพระดับสตูดิโอ ไม่สูญเสียรายละเอียด
WAV 32-bit Float Max Precision — ความแม่นยำสูงสุด เหมาะสำหรับนำไปประมวลผลต่อ
MP3 320kbps คุณภาพสูงสุดของ MP3 ไฟล์เล็กกว่า WAV มาก
MP3 256kbps คุณภาพดีมาก ไฟล์เล็กลง
MP3 192kbps ขนาดเล็กสุด เหมาะส่งต่อ/แชร์

เลือกฟอร์แมต → ดูขนาดไฟล์โดยประมาณ → กด Export! ใช้เวลาประมาณความยาวของเพลง

2. Reset Preset — คืนกลับไป Preset เดิม

เลือก Genre Preset แล้วปรับเพิ่มเติม แต่ไม่พอใจ? กดปุ่ม Reset Preset ที่ส่วน Mastering Controls ระบบจะคืนค่าทุก Slider กลับไปที่ Preset ของ Genre นั้นๆ ทันที ไม่ต้องเลือกใหม่!
⚠️ ถ้ายังไม่ได้เลือก Genre จะคืนเป็นค่า Default แทน

3. Reset EQ — คืนค่า EQ เป็น Flat

กดปุ่ม Reset EQ ที่อยู่ขวาสุดของ EQ Response Curve จะรีเซ็ต 7 Band EQ ทั้งหมดกลับเป็น 0 dB ทันที — เหมาะสำหรับเริ่มปรับใหม่จากศูนย์โดยไม่กระทบ Compressor หรือเอฟเฟกต์อื่น

4. Reset All — คืนค่าทุกอย่างเป็น Default

กดปุ่ม Reset All ที่ส่วน Mastering Controls จะรีเซ็ต ทุก Slider กลับเป็นค่า Default (ศูนย์ทั้งหมด) และยกเลิกการเลือก Genre ด้วย

5. A/B Compare — เปรียบเทียบเสียง

อยากเปรียบเทียบก่อน-หลัง? กด Stop แล้วปรับค่าใหม่ → กด Play ฟังใหม่ → ลองปรับอีก → ทำซ้ำจนกว่าจะพอใจ แล้วค่อย Export

6. ลองเปลี่ยน Genre Preset

ไม่พอใจผลลัพธ์? ลองเลือก Genre Preset อื่น ระบบจะปรับค่าทุกตัวให้เหมาะกับแนวเพลงนั้นอัตโนมัติ แล้วค่อยปรับแต่งเพิ่มเติมจากจุดนั้น ถ้าปรับเพิ่มแล้วไม่ชอบ กด Reset Preset ได้ไม่จำกัด

ปุ่ม Reset ทั้ง 3 แบบ

มีปุ่ม Reset ให้เลือก 3 แบบ แต่ละแบบทำงานต่างกัน — เลือกให้เหมาะกับสถานการณ์

🔄 Reset Preset — คืนกลับไป Preset เดิม

มีประโยชน์ที่สุด! เลือก Genre แล้วปรับเพิ่มเติม แต่ปรับไปปรับมาไม่พอใจ → กด Reset Preset ค่าทุก Slider จะกลับไปที่ Preset ของ Genre นั้นๆ ทันที
⚠️ ถ้ายังไม่ได้เลือก Genre จะคืนเป็นค่า Default แทน

📊 Reset EQ — คืนเฉพาะ EQ เป็น Flat

คืนค่า 7 Band EQ ทั้งหมดเป็น 0 dB โดยไม่กระทบ Compressor, Saturation, Stereo หรือเอฟเฟกต์อื่น — เหมาะเวลาอยากเริ่มปรับ EQ ใหม่จากศูนย์

🗑️ Reset All — คืนทุกอย่างเป็น Default

คืน ทุก Slider ทั้งหมด กลับเป็นค่า Default (ศูนย์ทั้งหมด) และยกเลิกการเลือก Genre ด้วย — เหมาะเวลาอยากเริ่มต้นใหม่ทั้งหมด

เปรียบเทียบปุ่ม Reset ทั้ง 3

ปุ่ม EQ Compressor เอฟเฟกต์อื่น Genre ที่เลือก
Reset Preset → Preset → Preset → Preset ✅ คงไว้
Reset EQ → 0 dB ไม่แตะ ไม่แตะ ✅ คงไว้
Reset All → 0 dB → Default → Default ❌ ลบ

วิธีใช้ EQ Drag Handles

ลากจุดกลมบน Curve

บน EQ Response Curve มีจุดกลม 7 จุด แทน Sub, Bass, Lo Mid, Mid, Hi Mid, Treble, Air — คลิกแล้วลากขึ้น-ลงได้เลย

Sync กับ Slider

เมื่อลาก Handle ค่าจะอัพเดทไปยัง Slider ด้านล่างทันที และในทางกลับกัน — ปรับ Slider แล้ว Handle บน Curve ก็จะขยับตาม

Hover ดูค่า

เอาเมาส์ใกล้ Handle จะแสดง Label ชื่อ Band + ค่า dB (เช่น "Bass +3.0 dB") ทำให้รู้ว่ากำลังปรับอะไร

รองรับ Touch

บนมือถือ ใช้นิ้วแตะที่จุดกลมแล้วลากขึ้น-ลงได้เลย ระบบจะจับตำแหน่งนิ้วแล้วแปลงเป็นค่า dB ให้อัตโนมัติ

Audio Signal Chain

Source Input Gain 7-Band EQ Compressor Make-Up Gain Saturation Stereo Width Limiter Noise Gate De-Esser Output Gain Analyser Destination
